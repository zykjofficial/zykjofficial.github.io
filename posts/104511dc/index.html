记安装黑苹果

发表于2020-11-14|更新于2021-01-13|黑苹果
|字数总计:937|阅读时长:3分钟

记第一次安装黑苹果

黑苹果: 自从苹果采用Intel的处理器，OS X被黑客破解后可以安装在Intel CPU与部分AMD CPU的机器上。从而出现了一大批非苹果设备而使用苹果操作系统的机器，被称为黑苹果(Hackintosh)；在Mac苹果机上面安装原版Mac系统的被称为白苹果（Macintosh），与黑苹果相对。

在去年暑假的时候、就想过安装黑苹果、当时看了许多的教程、但是没有找到适合我笔记本电脑的EFI、导致安装完成之后H630显存显示7M、声音也有点不正常、各种问题、然后第二次再进入系统时就卡在了网站、于是就只能放弃了(果然没有合适的EFI就是安装不成功)
在到今年寒假、又想尝试、然而卡在了最后两分钟、又再一次放弃。。。
直到最近、在群里看到有人成功安装了黑苹果、我又进入了黑果小兵、在其他机型中找到了适合我电脑的EFI F117B-Hackintosh、这又让我想安装黑苹果了、可是没有这么多的流量、好在同学那下载了镜像【黑果小兵】macOS Catalina 10.15.7 19H2 正式版 with Clover 5122原版镜像[双EFI版][UEFI and MBR]、这样我就开始了安装黑苹果之旅

准备工具：

  • 16G及以上U盘
  • macOS Catalina 10.15.7(19H2) Installer for Clover 5122 and WEPE.dmg
  • etcher
  • EFI文件、由于上面的EFI还是没用、就的其他的 https://www.chenweikang.top/?p=713
  • DiskGenius

https://blog.daliansky.net/MacOS-installation-tutorial-XiaoMi-Pro-installation-process-records.html

使用 etcher 格式化U盘

通过 DiskGenius 复制 EFI文件 进入 U盘的EFI分区里

进入Bios关闭安全模式、再切换成到U盘引导

最后按照上面的教程安装就行了

目前安装的黑苹果

  • 触控板可用
  • USB可用
  • 核显H630可用
  • 无线网卡无解(可能买个网卡)

记第二次安装黑苹果

时隔两个月、我再一次尝试安装黑苹果、也查考了很多视频教程和文字教程、在 Github 上找到了 https://github.com/chenchen6520/-F117B-OC-EFI 这个和我同机型的(使用的OC引导)。尝试安装、成功了。

成功安装上了Catalina，之后又尝试了BigSur，升级了OC引导后也安装成功了。

配置与驱动情况 （OpenCore 0.6、不支持BigSur）：

型号机械师 F117B2
处理器Intel(R) Core(TM) i7-8750H CPU @ 2.20GHz（完美驱动，变频正常)
集成显卡Intel UHD Graphics 630（完美驱动，4G 显存）
独立显卡GTX 1050Ti（自10.13.6起N卡无法驱动，HDMI、DP 依赖独显，亦未驱动）
无线网卡AC9462（Wi-Fi、蓝牙均完美驱动,不支持隔空投送）
运行内存16.00 GB（完美驱动）
机械硬盘HGST HTS541010B7E610 1TB（完美驱动） 、(没钱买固态、自带PM981又不支持)
内建屏幕1920x1080分辨率（完美驱动）
内建声卡Realtek ALC269（完美驱动）

其他驱动情况

  • 触控板完美驱动、支持手势

  • 支持安卓 USB 网络共享

  • 睡眠唤醒完全正常

  • 电源管理完美驱动

  • USB总线3.1、2.0完美驱动，包括后置 TypeC 触控板完美驱动

  • 支持部分手势 支持FN+1一键强冷 、FN+F1 睡眠、FN+F4 WIFI开关、音量和亮度控制

虽然折腾了这么久、但是我还是放弃了、毕竟还有诸多问题没有解决

如果你像折腾黑苹果的话、推荐几个网站：

  • Dortania’s OpenCore Install Guide

  • 黑苹果快速安装手册

  • OpenCore 简体中文参考手册

  • OC-little

应该不会再折腾黑苹果了
